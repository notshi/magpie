# The Magpie
The Magpie is Bradford's Radical News Sheet.

*We aim to publish twice a year but this is not set in stone.*  
However, we always publish one in time for the Bradford Zine Fair.

The Magpie is an â€˜occasional' publication - so not suitable for advertising immanent events.

Contributions usually have a theme but are always related to Bradford - with a humorous, radical or satirical edge.  
Articles can also be poems, photos and drawings. Please supply who to credit for articles.

Small: 50 - 100 words  
Quarter page: 300 - 500 words  
Half page: 600 - 1000 words  

Dimensions is double-sided A3 (297 x 420mm).  
Risographed in black monotone so please bear this in mind when sending artwork through.  
Printing is self-funded and each issue is estimated at 20-30 squids.  
Print run of 800 - 1000 copies at the Shipley Print Studio.  
Distribution across BD for free by HowDo?! Yorkshire.
